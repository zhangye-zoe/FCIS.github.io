Abstract

Cell instance segmentation is fundamental to analyzing biomedical images, yet accurately distinguishing tightly touching cells remains a persistent challenge. Existing methods balance model performance and computational efficiency imperfectly. We propose a novel segmentation framework inspired by the four-color theorem, assigning each cell one of four labels to guarantee adjacent instances differ. An asymptotic training strategy and encoding transformation ensure stable convergence. Extensive experiments demonstrate state-of-the-art performance on multiple biomedical image datasets.

Overview

By treating cells as “countries” and the background as the “ocean,” our four-color encoding reformulates instance segmentation into a four-class semantic task, simplifying post-processing and improving computational efficiency.

Method

Asymptotic Training: We first segment foreground vs. background, then predict the four-color labels within the foreground region.

Encoding Transformation: A learnable module resolves the non-uniqueness of greedy four-color assignments, mapping network outputs to a consistent minimum-color representation.

Negative Sampling: Orthogonality constraints between adjacent cell features enforce distinct color assignments.

Results

  • DSB2018: DICE = 0.939, PQ = 0.770
  • PanNuke: DICE = 0.816, PQ = 0.610
  • BBBC006v1: DICE = 0.954, PQ = 0.935

Our model outperforms state-of-the-art methods in both accuracy and efficiency, achieving superior segmentation quality with reduced parameters and FLOPs.
